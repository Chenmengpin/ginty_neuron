Differentially expressed genes

Using DESeq to evaluate differential expression between SA1 and Npy2r neuron subtypes, we get a total of 163 genes identifed as significant. This based on a somewhat stringent threshold of p-adjusted < 0.05 and abs(log2FoldChange) > 2.0. In the volcano plot depicted below, each of the genes are plotted with adusted p-value on the y-axis and log2FC on the x-axis. Purple points indicate those that meet significance criteria specified above.

# Extract contrasts
contrast <- list("neurontypeRA.LTMR" , "neurontypeSA1")
res <- results(dds, contrast=contrast)

# Set cutoffs
padj.cutoff <- 0.05
fc.cutoff <- 2

# Identify significant genes
res$threshold <- as.logical(res$padj < padj.cutoff & abs(res$log2FoldChange) > fc.cutoff)

# Volcano plots to visualize significant genes
df <- data.frame(res)
ggplot(data=df, aes(x=log2FoldChange, y=-log10(padj), colour=threshold)) +
  scale_color_manual(values = c("grey", "purple")) +
  xlim(c(-6,6)) +
  geom_point(alpha=0.75, pch=16) +
  theme(legend.position = "none",
        plot.title = element_text(size = rel(1.5)),
        axis.title = element_text(size = rel(1.5)),
        axis.text = element_text(size = rel(1.25))) +
  xlab("log2 fold change") + ylab("-log10 p-value")

Functional Analysis

We used the 163 genes as input to gProfileR, a tool that performs statistical enrichment analysis to find over-representation of information like Gene Ontology terms, biological pathways, regulatory DNA elements, human disease gene annotations, and protein-protein interaction networks. From the gprofiler results the significant Gene Ontolgy terms were extracted and some of the top terms are highlighted in the table below. The full results table containing all annotation terms can be found in the Files for Download section.

### Get gene names for only significant genes

The significant GO terms from gProfiler were used as input to REVIGO to visualize the most prominent processes and reduce redundancy of terms. Each rectangle is a single cluster representative, where terms sharing parent term at some level have been grouped together. The representatives are joined into ‘superclusters’ of loosely related terms, visualized with different colors. Size of the text reflect either the p-value, or the frequency of the GO term in the underlying GOA database.
